Basic quikFilter demo

This is a searchable list. Each element (<li> in this case) has an attribute called "tag" which is then use in jQuery later when activating quikFilter.


List:
  • Egg
  • Milk
  • Bread
  • Eggnog

How does it work?

quikFilter is enabled with jQuery using an event handler:

$("#searchfield").on("keyup", {search:"tag"}, quikFilter);

You can use any event handler to perform the filtering but .on() gives the best response. The 'search' attribute is the name of the attribute for which the quikFilter will be using to filter. It allows you to use custom attributes (such as 'tag', used in this demo) to filter elements. This means that you can use attributes like 'name', 'address' or 'flavour' - anything you want.
